# Student Experience Flow — Rua do Saber

## Core Principle
The learning experience is teacher-led and platform-supported.

The teacher controls checkpoints that unlock each stage of the lesson.

Teacher → Platform → Student

---

## Lesson Checkpoints

### Checkpoint 0 — Platform Introduction (Teacher)
Teacher introduces the platform to the class.
Students have not accessed the system yet.

Teacher sends invitation links to students via email.

### Checkpoint 1 — First Access Completed
Student accesses platform through invitation link.

Steps:
1. Create password
2. Choose nickname
3. Select avatar

This onboarding only happens once.

### Checkpoint 2 — Teacher Introduces Lesson Theme
Teacher explains the topic in class and answers questions.
After this explanation, the teacher unlocks the lesson.

### Checkpoint 3 — Video Completed
Student enters the journey map.
The first lesson node appears.

Mascot introduces the journey and connects the teacher explanation to the digital activity.

Student enters the cinema room and watches the animation.

### Checkpoint 4 — Reward Collected
After the video ends, a reward chest appears.
Student opens the chest and receives an animated collectible card.

### Checkpoint 5 — Challenge Completed
Mascot introduces the interactive challenge.

Possible formats:
- Memory game
- Puzzle
- Quiz
- Association game

Student plays until the XP bar fills and the level increases.

Student receives a new ability/status card.

### Checkpoint 6 — Optional Classroom Activity
Teacher may pause the digital experience and conduct a physical classroom activity.

### Checkpoint 7 — Mission Unlocked
Mascot introduces the mission.

Mission connects lesson with real world.

### Checkpoint 8 — Mission Validated
At the beginning of the next lesson the teacher validates missions.

The previous lesson cycle is completed.
